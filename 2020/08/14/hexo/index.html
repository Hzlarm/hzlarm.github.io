hexo

准备

安装hexo 之前需要安装Git 与Node.js

  • Git

新建一个公开仓库，仓库名格式为 .github.io name为github名字

  • Node.js

    安装 Node.js 的最佳方式是使用 nvm

安装hexo建立blog

部署

编辑配置文件，修改Deployment下的deploy字段

执行 hexo clean && hexo d -g

常用命令

hexo new XXX #新建文章，XXX可替换为你的文章名
hexo new page XXX #新建页面，XXX可替换为你的页面名
hexo generate #生成静态页面，可缩写hexo g
hexo server #开启预览，默认端口4000，可缩写hexo s
hexo deploy #部署到GitHub，可缩写hexo d
hexo help # 查看帮助
hexo version #查看Hexo的版本
